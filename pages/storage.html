Storage.

Primary Storage

Ram is a most popular example of primary storage. Random Access Memory (RAM) and allows the computer to use it as a buffer. It's purpose is to allow the computer to quickly access common files. This means the computer doesn't have to go through the slower storage to find something it constantly needs.

Secondary Storage

Secondary storage is usually mass storage. This means hard Disk drives (HDD) or Solid State Drives (SSD). Secondary storage nowadays have the ability to hold large amounts of data but trade off read and write speeds. This why the computer has both primary and secondary storage, to provide speed to common files while being able to store a lot of data.
