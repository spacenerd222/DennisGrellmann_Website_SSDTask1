What does the CPU do?

The fetch-execute cycle is the basic operation of a computer's central processing unit (CPU). It consists of two main steps:

  • The fetch step, where the CPU retrieves instructions from memory.
  • The execute step, where the CPU performs the instructions. In the fetch step, the CPU fetches the instruction from memory and stores it in its instruction register. Then, in the execute step, the CPU decodes and executes the instruction.
  • The result is stored in the appropriate register or memory location.
  • The cycle repeats, fetching the next instruction.

The fetch-execute cycle is crucial to the functioning of a computer, as it enables the CPU to execute programs and perform tasks.
